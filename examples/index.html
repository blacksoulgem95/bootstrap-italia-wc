Getting Started

Installation

I Web Components assumono che Bootstrap Italia CSS sia già importato globalmente. Include solo il JavaScript:

<!-- Bootstrap Italia CSS (richiesto - assumiamo che sia già importato globalmente) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-italia@2.0.0/dist/css/bootstrap-italia.min.css">

<!-- Bootstrap Italia WC JavaScript -->
<script src="path/to/bootstrap-italia-wc.js"></script>

Usage

Use the Web Components in your HTML:

<it-alert type="success" dismissible>
  <strong>Success!</strong> This is a success alert.
</it-alert>

API

All components support standard HTML attributes and custom properties. Check the individual example pages for detailed API documentation.
