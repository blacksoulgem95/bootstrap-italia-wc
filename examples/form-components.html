Bootstrap Italia WC - Form Components

Esempi di utilizzo dei componenti form di Bootstrap Italia implementati come Web Components.

Form Input

Componente per input di testo, email, password, numero, telefono, URL e ricerca.

Input di Testo

<it-form-input type="text" label="Nome" placeholder="Inserisci il tuo nome" help-text="Inserisci il tuo nome completo" required ></it-form-input>

Input Email

<it-form-input type="email" label="Email" placeholder="nome@esempio.it" validation="valid" validation-message="Email valida" ></it-form-input>

Input Password

<it-form-input type="password" label="Password" placeholder="Inserisci la password" required ></it-form-input>

Input Numero

<it-form-input type="number" label="Età" placeholder="Inserisci la tua età" min="18" max="100" required ></it-form-input>

Form Textarea

Componente per aree di testo multilinea.

Textarea Base

<it-form-textarea label="Messaggio" placeholder="Inserisci il tuo messaggio" rows="4" maxlength="500" help-text="Massimo 500 caratteri" required ></it-form-textarea>

Form Select

Componente per menu a tendina.

Select Base

<it-form-select label="Scegli un'opzione" required > <option value="">Seleziona...</option> <option value="1">Opzione 1</option> <option value="2">Opzione 2</option> <option value="3">Opzione 3</option> </it-form-select>

Select Multipla

<it-form-select label="Scegli più opzioni" multiple size="3" > <option value="1">Opzione 1</option> <option value="2">Opzione 2</option> <option value="3">Opzione 3</option> <option value="4">Opzione 4</option> </it-form-select>

Form Checkbox

Componente per checkbox.

Checkbox Base

<it-form-checkbox label="Accetto i termini e condizioni" value="accepted" required ></it-form-checkbox>

Checkbox con Validazione

<it-form-checkbox label="Abilita notifiche" value="notifications" validation="valid" validation-message="Notifiche abilitate" ></it-form-checkbox>

Form Radio

Componente per radio button.

Radio Button

<it-form-radio name="gender" label="Maschio" value="male" required ></it-form-radio> <it-form-radio name="gender" label="Femmina" value="female" ></it-form-radio> <it-form-radio name="gender" label="Altro" value="other" ></it-form-radio>

Form File

Componente per upload di file.

File Upload

<it-form-file label="Carica un file" accept=".pdf,.doc,.docx" help-text="Formati supportati: PDF, DOC, DOCX" required ></it-form-file>

File Upload Multiplo

<it-form-file label="Carica più file" accept="image/*" multiple help-text="Puoi selezionare più immagini" ></it-form-file>

Form Toggle

Componente per interruttori toggle.

Toggle Switch

<it-form-toggle label="Abilita notifiche" value="notifications" help-text="Ricevi notifiche via email" ></it-form-toggle>

Toggle con Validazione

<it-form-toggle label="Accetto i termini" value="terms" validation="invalid" validation-message="Devi accettare i termini" required ></it-form-toggle>

Form Group

Componente per raggruppare e validare più campi form.

Gruppo di Campi

<it-form-group label="Informazioni Personali" help-text="Inserisci le tue informazioni personali" > <it-form-input type="text" label="Nome" placeholder="Inserisci il nome" required ></it-form-input> <it-form-input type="text" label="Cognome" placeholder="Inserisci il cognome" required ></it-form-input> <it-form-input type="email" label="Email" placeholder="nome@esempio.it" required ></it-form-input> </it-form-group>
